# debug monitor

## install the package

    
    git clone https://github.com/dit-robotics/debugger.git
    
    cd ~/catkin_ws/src/debugger/src
    
    chmod +x debugger.py
    

## run the debug monitor

    
    rosrun debugger debugger.py
    

## It should report:

- txST1
- rxST1
- success_rate
- txST2
- rxST2
- success_rate_2
